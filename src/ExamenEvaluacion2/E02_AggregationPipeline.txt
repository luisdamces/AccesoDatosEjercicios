Nombre completo (firstName y lastName) de los jugadores en un solo campo y Edad actual en años de cada jugador.

db.jugadores.aggregate([{
    $project: {
        "nombreCompleto": {
            $concat: ["$firstName", " ", "$lastName"]
        },
        "dateOfBirthUTC": 1

    }
}, {
    $project: {
        "nombreCompleto": 1,
        "age": {
            $subtract: [2019, {
                $year: {
                    $toDate: "$dateOfBirthUTC"
                }
            }]
        }

    }
}])


Insertar el resultado en una nueva colección llamada, jugadores_edades.
